Shakespeare GPT

A basic implementation of a generative pre-trained transformer (also known as GPT) trained on an extrapolation of Shakespeare's play, "The tempest". I implemented a language model, encoder, token generator, loss estimator and optimizer. As part of the optimization, I implemented methods such as self-attention, feed-forward layer, residual connections and dropout. The GPT aims to generate and output that is similar to Shakespeare's play

NellieBot - Your personal NBA chat bot

A chat bot powered by LangChain that provides users with the most up-to-date scores and stats for NBA games. The chat bot is able to provide information on any game in the current and past seasons. It has a retrieval-augmented generation capability, allowing it to dynamically create a SQL query based on user input, execute it and return a natural-language answer to your question. Additionally, it has a memorization and contextualization ability, giving it an overview of all past messages, which enables to user to make queries without providing crucial information that has been provided previously
