Smart Brain

This is a project made with React and connected to a facial recognition API. The user must register and upon registration, they are able to enter a URL of an image, which gets displayed on the screen. If the image contains faces, the app will recognize them and put a box around them. The front end is connected to a server and a database, so that a user can access it at any time

Robofriends

This project was my first attempt at using React and connecting a project to an API. The users are fetched from a JSON placeholder website online and passed through an API that assigns each user to a random image of a robot based on their name. The user is able to search and scroll through the robots to find a specific one.

Password checker

This project was made in Python. The user enters a password which gets hashed using the SHA1 algorithm and the hash is split into the first 5 characters and the remainder of the hash. An API from "Have I been pwned?" is used to compare the passwords to their databases. Since the API implements k-anonimity, only the first 5 characters of the hash are compared using the API and the rest of the hash is compared locally to maintain security.
